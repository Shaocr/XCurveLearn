## Installation

You need the following packages to install XCurveOpt:
```python
- Python >= 3.6+
- Pytorch >= 1.8+
- Numpy >= 1.21+
- scikit-learn >= 1.0+
```
Then you can get XCurveOpt 
```sh
pip install XCurveOpt
```
